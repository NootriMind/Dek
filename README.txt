Files:

- assets/ folder with optimized JPG images

How to publish on GitHub Pages:
1. Create a new GitHub repository.
2. Upload index.html and the assets folder to the repository root.
3. Go to Settings > Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select the main branch and /root, then save.
6. GitHub will provide a public Pages URL.

How to publish on Netlify:
1. Go to https://app.netlify.com/drop
2. Drag the full folder into Netlify Drop.
3. Netlify will provide a shareable link.

Important:
Keep the assets folder next to index.html. If assets are moved, the photos will not load.
